# pi-tian-image-cache

Temporary pasted-image caching with compact placeholders for [pi](https://pi.dev).

```bash
pi install npm:pi-tian-image-cache
```

Provides `Ctrl+V` clipboard-image support on macOS, `/images`, and `/image-cache-clear`.

See the [collection repository](https://github.com/TianZuo555/pi-tian-extensions#image-cache) for full documentation.
