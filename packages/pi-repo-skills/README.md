# pi-tian-repo-skills

Per-repository skill enable and disable controls for [pi](https://pi.dev).

```bash
pi install npm:pi-tian-repo-skills
```

Provides `/skills`, `/skills-list`, `/skills-reset`, and the `repo_skills` tool.

See the [collection repository](https://github.com/TianZuo555/pi-tian-extensions#pi-repo-skills) for full documentation.
